Features

Everything you need to redact with confidence

Built on Microsoft Presidio and spaCy — the same NLP stack trusted by enterprise security teams.

🔒
Fully offline

Once installed, the app requires no internet connection. Models run locally via spaCy. No document data is ever transmitted anywhere.

PRIVACY-FIRST
🧠
AI-powered PII detection

Microsoft Presidio with spaCy's en_core_web_lg model detects names, organisations, dates, phone numbers, IDs, and more — with confidence scores.

NLP · NER
✅
Review before you redact

Every detected entity is listed for your review. Approve, reject, or remove individual items before a single redaction is applied.

HUMAN-IN-THE-LOOP
✏️
Manual phrase redaction

Type any phrase to add it to the redaction list — useful for domain-specific identifiers the AI may not recognise, like internal project codes.

CUSTOM PHRASES
🖊️
Draw-to-redact on PDFs

Switch into drawing mode and drag black boxes directly over any area of a PDF page — perfect for redacting signatures, stamps, or images that text search cannot find.

Works with PDF (text-based), DOCX, and TXT files. PDF redactions are applied as permanent black boxes using PyMuPDF — no recoverable text underneath.

How it works

From raw document to redacted file in four steps

The entire workflow runs in your browser tab, served locally. Nothing touches the internet.

Install once

Download and unzip. Run the setup script to create a local Python environment and download the AI model.

setup.bat
Launch the app

Double-click to start. Your browser opens automatically at localhost:8501 — no internet required.

run.bat
Upload & review

Upload your PDF, DOCX, or TXT file. The AI scans it and presents every detected entity for your approval. Accept, reject, or add your own phrases.

Apply & download

Click Apply Redactions. Draw any additional black boxes over images or signatures. Download your clean, redacted document.

Detection coverage

What gets detected

Out of the box, the engine detects the following PII entity types. Custom phrases can supplement detection for anything not listed.

PERSON ORGANIZATION LOCATION DATE_TIME PHONE_NUMBER EMAIL_ADDRESS CREDIT_CARD US_SSN IP_ADDRESS CONTRACT_ID EIN BANK_ROUTING NRIC_SG ADDRESS + custom phrases

System requirements

What you need

Lightweight. No GPU required. The setup script handles everything after step one.

  • Windows 10 or 11 (64-bit)
  • Python 3.10, 3.11, or 3.12 — install from python.org, tick "Add Python to PATH" during setup
  • ~1.5 GB free disk space (Python environment + spaCy model)
  • Internet access during first-time setup only — to download dependencies
  • Any modern browser (Chrome, Edge, Firefox) — opens automatically on launch
